趋势符号，标记上升和下降状态。

## API

```html
<Trend flag="up">50%</Trend>
```

| 参数      | 说明                                      | 类型         | 默认值 |
|----------|------------------------------------------|-------------|-------|
| colorful | 是否彩色标记 | Boolean | true |
| flag | 上升下降标识：`up\|down` | string | - |
